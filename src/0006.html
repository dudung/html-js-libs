# draw rotating box in HTML using Three.js
## result
## code Previous result is obtained using
```html <script> const camera = new THREE.PerspectiveCamera( 70, window.innerWidth / window.innerHeight, 0.01, 10 ); camera.position.z = 1; const scene = new THREE.Scene(); const geometry = new THREE.BoxGeometry( 0.5, 0.5, 0.5 ); const material = new THREE.MeshNormalMaterial(); const mesh = new THREE.Mesh( geometry, material ); scene.add( mesh ); const renderer = new THREE.WebGLRenderer( { antialias: true } ); renderer.setSize( 0.4 * window.innerWidth, 0.4 * window.innerHeight ); renderer.setAnimationLoop( animation ); document.body.appendChild( renderer.domElement ); // animation function animation( time ) { mesh.rotation.x = time / 2000; mesh.rotation.y = time / 1000; renderer.render( scene, camera ); } ```
in body section and with
```html <script src="https://unpkg.com/three@0.146.0/build/three.min.js"> ```
in head section of a HTML file.
